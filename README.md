# 💰 MERN Expense Tracker

A full-stack **Expense Tracker Web Application** built using the **MERN stack (MongoDB, Express.js, React.js, Node.js)**.
This application helps users efficiently track income and expenses, manage transactions, and visualize their financial activity in a simple and intuitive interface.

---

## ✨ Features

* ➕ Add income and expense transactions
* ❌ Delete transactions
* 📊 View real-time balance, total income, and total expenses
* 🧾 Transaction history management
* 📱 Fully responsive design
* ⚡ Fast and lightweight performance

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Axios
* CSS / Tailwind / Bootstrap (based on your implementation)

### Backend

* Node.js
* Express.js
* MongoDB + Mongoose


## 🚀 Getting Started

Follow these instructions to set up the project locally.

---

## 📦 Prerequisites

Make sure you have installed:

* Node.js
* npm or yarn
* MongoDB (local or Atlas)

---

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/expense-tracker.git
cd expense-tracker
```

---

### 2. Backend Setup

```bash
cd backend
npm install
```

Create a `.env` file in the backend folder:

```
PORT=5000
MONGO_URI=your_mongodb_connection_string
```

### 3. Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Create a `.env` file in the frontend folder:

```
REACT_APP_API_URL=http://localhost:5000
```

Start the frontend:

```bash
npm start
```
